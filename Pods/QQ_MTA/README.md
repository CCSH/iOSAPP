# MTA
MTA 官方CocoaPods Spec镜像

## 使用方法
### 基础统计功能
Podfile 中添加
```
pod 'QQ_MTA'
```
__注意: 为了提高统计的精确性, IDFA 模块已经与 MTA 模块合并, 为避免 IDFA 模块影响 APP上架, 请参考 [idfa 配置文档](https://mta.qq.com/docs/iOS_idfa.html)进行相关配置__

### H5 App 混合统计功能
Podfile 中添加
```
pod 'QQ_MTA/Hybrid'
```
