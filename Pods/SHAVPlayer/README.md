# SHAVPlayer
视频播放器
# pod导入
```
pod 'SHAVPlayer'
```
- [x] 音频播放
- [x] 视频播放
- [x] 本地播放、网络播放、边下边播
- [x] 缓存进度
- [x] 播放进度
- [x] 自动播放
- [x] 音频后台播放
- [x] 全屏播放(demo中有方法)
- [x] 拖拽进度
- [x] 锁屏显示
- [x] ios 9.1 以上支持 锁屏拖拽进度播放
- [x] 耳机控制
- [x] 锁屏控制 播放、暂停、下一首、上一首等
- [x] 加载中状态监听
