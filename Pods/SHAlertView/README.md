# SHAlertView
# 使用 pod 导入
[![Pod Version](http://img.shields.io/cocoapods/v/SHAlertView.svg?style=flat)](http://cocoadocs.org/docsets/SHAlertView/)
```
pod 'SHAlertView'
```
- [x] 标准（支持 字符串与富文本）
- [x] 富文本局部文字可点击
- [x] 插画
- [x] 图标
- [x] 图标 + 自定义视图
- [x] 自定义视图
